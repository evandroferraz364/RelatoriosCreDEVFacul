# Relatório de Impulsionamento | Meta Ads

Aplicativo web para processar exportações do Meta Ads e gerar relatórios em Excel.

## Recursos
- Upload de planilhas `.xlsx` exportadas do Meta Ads
- Detecção automática de cliente e período
- Classificação automática de campanhas
- Revisão manual antes da exportação
- Preenchimento da capa do relatório
- Exportação do Excel final

## Publicação no GitHub Pages
1. Envie os arquivos para a raiz do repositório
2. Vá em **Settings > Pages**
3. Em **Source**, escolha **Deploy from a branch**
4. Selecione a branch **main** e a pasta **/ (root)**
5. Salve
